Given `` n `` pairs of parentheses, write a function to _generate all combinations of well-formed parentheses_.

&nbsp;

__Example 1:__

<pre><strong>Input:</strong> n = 3
<strong>Output:</strong> ["((()))","(()())","(())()","()(())","()()()"]
</pre>

__Example 2:__

<pre><strong>Input:</strong> n = 1
<strong>Output:</strong> ["()"]
</pre>

&nbsp;

__Constraints:__

*   `` 1 &lt;= n &lt;= 8 ``
